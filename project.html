Data Visualization Project

Langauges: HTML,CSS, javascript,D3.js

Zillow Data US Home Values

This project aims at visualizing Home pricing trends in United States for the past 23 years and also giving brief insight of states and county level home pricings as of today. This is interactive visualization project which allows users to interact with visualization at every step. Allowing the user to select filters and drill down into data from particular states/regions.

Description of Data

Zillow is an American online real estate database company. It provides datasets publicly on their website for research purposes. The data used for this project is US home values from 1996 to 2019. It is 25mb data set with 287 columns 13237 rows.
